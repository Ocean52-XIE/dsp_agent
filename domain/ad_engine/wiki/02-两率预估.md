# 两率预估（pCTR / pCVR）

## 1. 职责

两率预估用于估计候选广告被点击与转化的概率，是出价和精排的核心输入。

## 2. 常见模型形态

- pCTR：点击率模型，常用于估计点击概率。
- pCVR：转化率模型，常用于估计成交或深转化概率。
- 多任务模型：共享底层特征并同时预测 CTR、CVR。

## 3. 特征体系

### 3.1 用户侧

- 用户画像特征（兴趣、消费层级）。
- 历史行为特征（曝光、点击、转化序列）。

### 3.2 广告侧

- 创意特征（标题、素材 embedding）。
- 计划特征（目标事件、预算、出价模式）。

### 3.3 上下文侧

- 广告位、页面场景、时段、网络环境。

## 4. 关键指标

- AUC：区分能力。
- LogLoss：概率预测质量。
- Calibration Error：概率校准误差。
- 线上 CTR/CVR 偏差：预测与真实的偏差稳定性。

## 5. 常见问题与处理

### 5.1 pCTR/pCVR 集体偏高或偏低

重点排查：

- 特征分布是否发生漂移。
- 模型版本是否误切换。
- 校准参数是否同步。

### 5.2 冷启动广告无量

处理思路：

- 引入探索流量配额。
- 使用内容相似召回 + 类目先验。
- 对冷启动计划使用分层保底策略。

## 6. 与出价联动

常见 eCPM 形态：

`eCPM = pCTR * Bid * 1000`

当目标是转化时也常见：

`eCPM = pCTR * pCVR * GoalValue * 1000`

注意：

- 两率预测若未做校准，会直接放大出价误差。
- 出价策略变更前，需先确认两率模型稳定性。
