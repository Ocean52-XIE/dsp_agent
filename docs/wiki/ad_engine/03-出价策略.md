# 出价策略

## 1. 目标

出价策略负责将广告主目标（CTR、CPA、ROI、GMV）映射为可参与竞价的实时出价。

## 2. 常见模式

- CPC：按点击计费。
- CPM：按千次曝光计费。
- oCPC：以转化目标约束 CPC。
- tCPA：目标单次转化成本。
- tROAS：目标投入产出比。

## 3. 实时出价核心要素

- 预估效果：pCTR、pCVR。
- 目标参数：target_cpa、target_roas。
- 预算状态：剩余预算、预算消耗速度。
- 风险参数：频控、异常流量过滤、计划限速。

## 4. 常见公式示意

在转化目标下，常见简化形式：

`bid = target_cpa * pCVR * alpha`

其中：

- `alpha`：策略调节因子，受预算进度和竞争强度影响。

## 5. 预算与节奏（Pacing）

- 预算超前消耗：下调调节因子，限制高价竞拍。
- 预算落后消耗：上调调节因子，增加拿量能力。
- 强烈建议按小时维度做动态 pacing，而不是静态日均分配。

## 6. 常见线上问题

### 6.1 胜率下降

排查：

- 竞对出价是否明显抬升。
- 两率预估是否偏低导致 eCPM 下降。
- 预算 pacing 是否过于保守。

### 6.2 成本超标

排查：

- 目标 CPA/ROI 配置是否错误。
- 转化回传延迟是否导致反馈滞后。
- 是否缺少高风险流量抑制规则。

## 7. 工程实践

- 出价模块需记录“输入特征 + 目标配置 + 计算结果 + 截断原因”。
- 所有调参应支持灰度开关与回滚。
- 出价日志要可追溯到请求 trace_id 与广告计划 ID。
